Quien quiere ser millonario — Documentación del Juego

Pantallas
- Pantalla principal
- Pantalla de configuración
- Pantalla de vídeo de introducción
- Pantalla de preguntas
- Pantalla de ranking (opcional)
- Pantalla final de fin de partida

Funcionamiento general
El objetivo es superar 10 preguntas seguidas, cada una un poco más difícil que la anterior.
Para cada número de pregunta hay 50 preguntas posibles. El juego elige una al azar.
Las preguntas tienen 4 opciones: A, B, C y D, siempre mezcladas antes de mostrarse.

Puntuación y tiempo
- Cada acierto suma 100 puntos.
- Cada comodín usado resta 100 puntos.
- Cada pregunta dispone de 15 segundos salvo si se usa el comodín de tiempo.

Pantalla principal
Incluye:
- Nueva partida
- Configuración
- Ranking (opcional)

Nueva partida
1. Se muestra un vídeo de introducción.
2. Se pide el nombre del jugador.
3. Comienza la primera pregunta.

Preguntas
El juego tiene 10 preguntas. Para cada una hay 50 variantes posibles.
La pregunta aparece centrada en tarjeta y debajo las respuestas A, B, C y D.
Las respuestas siempre se mezclan.

Temporizador
Cada pregunta tiene 15 segundos. Si se agota el tiempo se falla automáticamente.

Corrección
- Si acierta: botón en verde, sonido positivo y +100 puntos.
- Si falla: botón en rojo, sonido de fallo y termina la partida.
Luego se muestra la pantalla final.

Puntuación
Se empieza con 0 puntos. Aciertos suman 100. Cada comodín resta 100.

Comodines (uno de cada por partida)
1. Comodín 50%: elimina dos opciones falsas y cuesta 100 puntos.
2. Comodín de tiempo: añade 1 minuto en la pregunta actual y cuesta 100 puntos.
3. Comodín de cambio de pregunta: cambia la pregunta por otra del mismo nivel y cuesta 100 puntos.

Pantalla de configuración
Permite activar o desactivar sonido. Se guarda en localStorage.
Afecta a efectos, música, fallos y aciertos.

Ranking (opcional)
Muestra partidas jugadas en ese dispositivo:
- Nombre
- Máxima pregunta alcanzada
- Puntos finales
Ordenado de mayor a menor.

Fin de partida
